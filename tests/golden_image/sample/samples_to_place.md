### Files to Generate (High Priority)

```bash
# BMP
convert -size 50x50 xc:purple BMP3:test.bmp

# RAR
echo "test content for rar" > /tmp/test.txt
rar a test.rar /tmp/test.txt

# 7z
echo "test content for 7z" > /tmp/test.txt
7z a test.7z /tmp/test.txt

# PPTX (use LibreOffice or download)
# Option: convert existing PPT to PPTX

# SQLite with browser-like schema
sqlite3 test.sqlite "
CREATE TABLE urls(id INTEGER PRIMARY KEY, url TEXT, title TEXT, visit_count INTEGER, last_visit_time INTEGER);
CREATE TABLE visits(id INTEGER PRIMARY KEY, url INTEGER, visit_time INTEGER);
INSERT INTO urls VALUES(1, 'https://example.com/', 'Example', 5, 13350000000000000);
INSERT INTO urls VALUES(2, 'https://test.org/page', 'Test Page', 3, 13350000000000000);
"

# strings.txt
cat > strings.txt << 'EOF'
```

### Size Concerns

Total current samples: **~15-20 MB** (too large for git!)

**Recommendation:** 
1. Use smallest variants (100KB JPG, 500KB PNG, etc.)
2. Generate tiny versions of large files
3. Target total raw image size: **< 5 MB**
4. E01 compressed: **< 2 MB**
